# Favicon и Icons

В этой папке должны находиться favicon и иконки для сайта.

## Необходимые файлы:

1. **favicon.ico** - Главный favicon (32x32px)
2. **favicon-16x16.png** - Для Safari (16x16px)
3. **favicon-32x32.png** - Для Windows (32x32px)
4. **apple-touch-icon.png** - Для iOS (180x180px)
5. **icon-192x192.png** - Для Android PWA (192x192px)
6. **icon-512x512.png** - Для Android PWA (512x512px)
7. **og-image.jpg** - Open Graph изображение (1200x630px)

## Инструменты для создания:

- **Favicon Generator**: https://favicon.io/
- **RealFaviconGenerator**: https://realfavicongenerator.net/
- **Favicon Converter**: https://convertico.com/

## Цветовая палитра:

- Primary: #EF4444 (red)
- Secondary: #8B5CF6 (purple)
- Background: #000000

## Создание favicon:

1. Создайте логотип 512x512px
2. Загрузите на https://favicon.io/favicon-generator/
3. Скачайте готовый пакет favicon
4. Распакуйте файлы в папку public/

## Создание OG изображения:

1. Создайте изображение 1200x630px
2. Добавьте логотип, заголовок и description
3. Используйте брендовые цвета
4. Сохраните как og-image.jpg
